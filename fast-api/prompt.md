# Prompt for Generating Graph DSA Topics

Generate a complete, well-structured learning roadmap for **Graph Data Structures and Algorithms** tailored for a **Software Developer Intern** role.

Please organize the content as follows:

## 1. Foundational Concepts

- Graph terminology (vertices, edges, directed vs undirected, weighted vs unweighted)
- Graph representations (adjacency matrix, adjacency list, edge list)
- Time and space complexity analysis for different representations
- When to use which representation

## 2. Graph Traversal Algorithms

- Breadth-First Search (BFS)
    - Implementation details
    - Applications and use cases
    - Time and space complexity
- Depth-First Search (DFS)
    - Implementation details
    - Applications and use cases
    - Time and space complexity
- Variations and optimizations

## 3. Shortest Path Algorithms

- Dijkstra's Algorithm (non-negative weights)
- Bellman-Ford Algorithm (handles negative weights)
- Floyd-Warshall Algorithm (all-pairs shortest path)
- A* Search Algorithm (heuristic-based)
- When to use which algorithm

## 4. Minimum Spanning Tree

- Kruskal's Algorithm
- Prim's Algorithm
- Union-Find data structure
- Real-world applications

## 5. Advanced Graph Algorithms

- Topological Sort
- Strongly Connected Components (Kosaraju's and Tarjan's algorithms)
- Detecting cycles in graphs
- Bipartite graph checking
- Bridges and articulation points

## 6. Special Graph Types

- Trees as graphs
- Directed Acyclic Graphs (DAGs)
- Bipartite graphs
- Complete graphs
- Weighted graphs

## 7. Common Interview Problems

For each category above, provide:

- 3-5 LeetCode-style problems (easy to medium difficulty)
- Problem-solving patterns specific to graph problems
- Common pitfalls and debugging tips

## 8. Implementation Practice

- Template code for each major algorithm
- Best practices for graph problem-solving
- Code optimization techniques

## 9. Real-World Applications

- Social networks
- Maps and navigation systems
- Network routing
- Dependency resolution
- Web crawling

## 10. Learning Resources

- Recommended practice platforms
- Video tutorials
- Books and articles
- Interview preparation tips

**Format Requirements:**

- Use clear headings and subheadings
- Include time complexity and space complexity for each algorithm
- Provide pseudocode or implementation hints where helpful
- Organize from beginner to advanced topics
- Include a suggested learning timeline (e.g., "Week 1: Foundations", etc.)
